# Features
• Revamped Controller Help Hints
Adds controller help hint buttons for every Bloopair controller, as well as every official controller's colored buttons (e.g. Red Wii Remote)
• Secondary Screen
Adds inventory management, crafting, and map to be shown kn the Wii U gamepad while normal gameplay remains on TV
• Dual Screen
Allows 1-4 people to play on TV, while one person can play on the gamepad
• Wii Remote Support
Allows player to use a Wii Remote to play, as long as the Nunchuk is attached. Uses advantage of the Wii Remote's motion controls, such as shaking the Wii Remote or Nunchuk.
•Gamecube controller support
Allows player to use the Gamecube controller, possible support for the Gamecube-Keyboard controller later on.
# Experimental Features
• 5 players
Allows 4 people to play on the TV, while one other plays on the gamepad.
• Adjustable Splitscreen
Allows players to pick where there screen is, picking between top left corner, top right corner, bottom left corner, bottom right corner, top half, bottom half, left half, right half, on TV, with toggle-able Secondary Screen option if playing on gamepad. And also a choice to just play on the gamepad.
• Map on TV
Allows players to choose if the map takes up a space of the TV, making use of the Adjustable Splitscreen feature to choose what section.
• Cross-Platform
Not much information is known for this feature, but it will either be cross platform between Wii U and Java Edition, Wii U and other LCE games, or both.
